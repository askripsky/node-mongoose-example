This is a simple example of using Mongoose.  [Mockgoose](https://www.npmjs.org/package/mockgoose) is used for in-memory testing.

The sample application creates a record, finds it and deletes it.

## Running Tests

      npm install
      grunt

## Running Application

      node app.js
